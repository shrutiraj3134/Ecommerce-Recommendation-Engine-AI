# NexCart Engine AI Pro 🛒
### Advanced ML Recommendation System & Analytics Dashboard

An end-to-end Applied Machine Learning microservice built using Python, Scikit-Learn, and Streamlit. The application processes real-world marketplace data to resolve cold-start issues, remove descriptive inconsistencies, and provide real-time cluster-based product predictions.

---

## 🚀 Key Features & Architectural Framework
- **Data Engineering Pipeline:** Real-time missing value treatment, programmatic feature extraction, and high-dimensional User-Item Matrix compression into CSR format.
- **Machine Learning Core:** Unsupervised **K-Nearest Neighbors (KNN)** clustering engine utilizing **Cosine Proximity Proximity** to identify exact identical peer consumers.
- **Anti-Bias Fallback Engine:** Features a dynamic global marketplace trend analyzer to gracefully combat data sparsity and class-imbalance anomalies.
- **Interactive UI & Analytics Space:** Dual-column operational space featuring distribution charts mapping active marketplace review frequencies.

---

## 🛠️ Tech Stack & Concepts Deployed
- **Language:** Python
- **Core Libraries:** Pandas, NumPy, Scikit-Learn, SciPy
- **UI Framework:** Streamlit (HTML/CSS Custom Injections)
- **Algorithms:** Unsupervised K-Nearest Neighbors (KNN), Cosine Similarity Matrix

---

## 📁 Repository Directory Architecture
- `data/` : Host directory for the processed marketplace product reviews transaction logs.
- `notebooks/` : Interactive prototyping workspace documenting exploratory data analysis (EDA) and experimental pipeline models.
- `app/` : Production ready scripts hosting custom styled layouts and inferencing execution logic.
